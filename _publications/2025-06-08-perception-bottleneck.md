This paper investigates the perception bottleneck in Vision-Language Models (VLMs) for chart understanding, identifying key limitations in how VLMs process and interpret visual information from charts.

Co-authors: Weihao Zeng, Xiwen Zhang, Yijun Wang, Zifei Shan, Junxian He.

Has GitHub code repository: Vision4Chart.
